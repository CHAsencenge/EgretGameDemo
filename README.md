# EgretGameDemo

#### EgeretGameDemo为一个新手上手Egret引擎的Demo项目，旨在：

从轻量级的Demo中逐渐熟悉TypeScript

了解监听事件、碰撞检测、遮罩、延迟触发、舞台、Tween等基本概念

练习使用Egret常用类库提供的功能
